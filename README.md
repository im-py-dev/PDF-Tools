- [X] Pdf To Text
- [X] Text To Pdf
- [X] Pdf Merger
- [X] Pdf Spliter
- [X] Pdf Rotater
- [X] Pdf Watermarker

- [X] Pdf Decrypter

## INSTALLTION:

``` 
apt update
apt upgrade
git clone https://github.com/impydev/PDF-Tools
cd PDF-TOOLBOX
pip3 install -r requirements.txt
python3 pdf-toolbox.py
```
